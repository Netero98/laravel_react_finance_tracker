PWA Test Page

How to Test Your PWA

Follow these steps to verify that your Progressive Web App is working correctly:

1. Check Service Worker Registration

  • Open Chrome DevTools (F12 or Right-click > Inspect)
  • Go to the "Application" tab
  • Select "Service Workers" in the left sidebar
  • Verify that your service worker is registered and active

2. Verify Manifest

  • In Chrome DevTools, go to the "Application" tab
  • Select "Manifest" in the left sidebar
  • Check that your manifest is properly loaded and parsed
  • Verify that all icons are loading correctly

3. Test Offline Functionality

  • In Chrome DevTools, go to the "Network" tab
  • Check the "Offline" checkbox to simulate offline mode
  • Refresh the page and verify that it still loads from the cache
  • Navigate to different pages to test offline navigation

4. Test Installation

  • On Chrome desktop, look for the install icon in the address bar
  • On mobile devices, you should see an "Add to Home Screen" prompt
  • Try installing the app and launching it from your home screen

PWA Audit

For a comprehensive check of your PWA, use Lighthouse:

  • In Chrome DevTools, go to the "Lighthouse" tab
  • Check the "Progressive Web App" category
  • Click "Generate report"
  • Review the results and address any issues
